To run this project a prerequisite is to have a .env files that contains a working Gemini API in the format of: APIKEY = (INSERT_API_KEY_HERE)

For optimal usage, initalize a Virtual Environment that is able to run the Flask program isolated form other project dependencies

While your VENV is running 

1. install the requirements

pip install -r requirements.txt

2. run the program for intialization with

python app.py

3. put the desired files into the input folder that has been created

4. open and modify the "config.json" file inputing the desired tags in the same format of the "example#" tags (Try to be discriptive yet consise)

5. REMEMBER to save the config file befor running the accessing the link and starting the run
